# SQL-PL-SQL-NoSQL

Repositorio de casos prácticos y ejercicios de SQL, PL/SQL y NoSQL.

## Estructura del Repositorio

### Casos Individuales

- **01-Caso-ANID**: Sistema de becas de especialización médica (PL/SQL)
- **02-Caso-Chinook**: Sistema de repuestos y servicios automotrices (NoSQL/MongoDB)

### MultiCasos

- **01-Truck_Rental**: Arriendo de camiones y maquinarias
- **02-Renta_a_House**: Corredora de propiedades
- **03-All_The_Best**: Empresa de retail con servicios financieros
- **04-Bank_Solutions**: Institución financiera
- **05-Ketekura**: Clínica de salud privada
- **06-IPFuturo**: Instituto profesional
- **07-Caso-Kopera**: Cooperativa de ahorro y crédito

## Estructura de Cada Caso

```
Caso/
├── Caso_*.md          # Contexto y requerimientos
├── DB/                # Scripts de creación y poblado de tablas
└── SQL/               # Soluciones SQL
    └── solucion.sql
```

## Tecnologías

- **SQL**: Oracle Database
- **PL/SQL**: Procedimientos, funciones, triggers y packages
- **NoSQL**: MongoDB
